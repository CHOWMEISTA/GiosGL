GiosGL Documentation

GiosGL is a zero-dependency, high-performance WebGL2 library heavily optimized for 4K displays and real-time computation (GPGPU).

Initialization

new GiosGL(canvas)

Initializes the engine. Automatically applies `desynchronized: true`, disables `antialias` for 4K performance, and sets internal resolution to 3840x2160.

resize(width, height)

Resizes the WebGL drawing buffer and updates the viewport. Fires all registered onResize listeners.

run(drawCallback)

Starts the main animation loop. drawCallback(time, delta) is called every frame with time measured in seconds.

Shader System

createProgram(id, vsSrc, fsSrc, attribLocations)

Compiles vertex and fragment shaders into a program. Caches all active uniforms and attributes for O(1) lookups during the render loop.

useProgram(id)

Binds the specified program for current rendering.

setUniform[1f|2f|3f|4f|1i|Matrix4fv](name, value)

Sets a uniform on the currently active program. Extremely fast due to internal caching.

Buffers & VAOs

createVBO(data, id?, usage?)

Uploads a Float32Array to the GPU as an ARRAY_BUFFER.

createVAO(attribs, ibo?, id?)

Creates a Vertex Array Object. attribs is an array of objects specifying {vbo, size, type, normalized, stride, offset, location, divisor?}.

Textures & Framebuffers

createTexture2D(id, w, h, opts)

Creates a 2D texture. Supports options for format, internalFormat, type, and pixels. Defaults to NEAREST filtering.

createFramebuffer(id, opts)

Generates an FBO. Pass { texture: myWebGLTexture } to bind a color attachment. Pass { depth: true } to auto-generate a depth renderbuffer.

blitFramebuffer(src, dst, w, h, mask, filter)

High-speed pixel copy from one FBO to another. Ideal for post-processing chains.

GPU Timing Queries

Measure exact hardware execution time using EXT_disjoint_timer_query_webgl2.

beginTimerQuery(name) / endTimerQuery(name)

Wrap your draw calls in these to measure execution time. Retrieve results frames later using resolveTimerQuery(name).

Event Handling

on(eventType, handler)

Binds standard DOM events (mousedown, touchmove, wheel) natively through the engine.

pickPixel(x, y, width, height)

Reads pixels back to the CPU via gl.readPixels. Perfect for color-based object picking.

Math Utilities

GiosGL includes static math helpers to keep the library dependency-free.
