GiosGL v1.0.0

Welcome to the official documentation for GiosGL, the world's most lightweight 4K-native WebGL2 engine. GiosGL is designed for developers who need extreme performance, low latency, and professional-grade GPU control without the overhead of heavy frameworks.

Core Philosophy

  • 4K Everywhere: Native 3840x2160 internal scaling.
  • Zero Garbage: Designed for 60FPS with minimal allocation.
  • GPGPU First: First-class support for simulation and compute-like tasks.
// Installation
const engine = new GiosGL(document.getElementById('canvas'));
engine.resize(3840, 2160); // Optional: Default is 4K

API Reference

engine.createProgram(id, vs, fs, attribs)

Compiles and links a new shader program. Automatically caches all uniform locations for O(1) performance.

engine.createVAO(attribs, ibo, id)

Configures a Vertex Array Object. Maps VBOs to shader attributes with zero manual binding required in the loop.

engine.setUniformMatrix4fv(name, mat4)

High-speed matrix upload. Uses cached locations to avoid string-parsing overhead during frames.

Advanced GPGPU

GiosGL excels at stateful simulations (Falling Sand, Fluids, Cellular Automata) by utilizing texture-based compute loops.

Ping-Pong Rendering

Use framebuffers and textures to feed the GPU's output back into itself as input.

// Example Ping-Pong
engine.createFramebuffer('bufferA', { texture: texA });
engine.createFramebuffer('bufferB', { texture: texB });

// In loop
engine.bindFramebuffer('bufferB');
engine.bindTexture2D('texA', 0); // Use previous state
engine.drawTriangles(4);
engine.blitFramebuffer(src, dst, w, h)

Performs a high-speed hardware copy from one buffer to another, essential for post-processing effects.

Utilities & Profiling

engine.beginTimerQuery(name)

Starts a hardware-level GPU timer query. Measures execution time in nanoseconds on the actual GPU silicon.

engine.saveScreenshot(filename)

Triggers a download of the current 4K frame buffer as a high-resolution PNG.

Math Helpers

GiosGL provides static math utilities to avoid external dependencies like glMatrix.

const projection = GiosGL.mat4Perspective(45, 16/9, 0.1, 1000);
const view = GiosGL.mat4LookAt([0,0,5], [0,0,0], [0,1,0]);
